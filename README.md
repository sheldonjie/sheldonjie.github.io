### OSI Reference Model(开放系统互连参考模型)


> 参考模型：是指该模型不是必须执行的标准，而是按照该模型的要求，不同企业生产的网络设备可以互相连接。


**通常将网络功能分配到不同层实现：**
- 高层使用底层的服务
- 层与层之间相对独立
- 通过SAP进行数据交换
